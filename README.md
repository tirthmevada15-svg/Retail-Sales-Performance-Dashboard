# Retail Sales Performance Dashboard

## Project Overview
The Retail Sales Performance Dashboard is a data analytics project designed to extract, analyze, and visualize retail transaction data using **SQL** and **Power BI**. The objective is to generate actionable insights that support business decision-making related to sales performance, customer behavior, and product trends.

## Dataset Description
The dataset contains **20,000 retail transactions** with the following key fields:
- TransactionID
- TransactionDate
- CustomerID
- ProductCategory
- Quantity
- UnitPrice
- PaymentMethod
- City
- TotalAmount

The data is clean, consistent, and suitable for analytical processing.

## SQL Analysis
SQL was used to:
- Validate and explore the dataset
- Aggregate sales and customer metrics
- Create reusable SQL views for dashboarding

### Key SQL Outputs:
- City-wise sales (last quarter)
- Monthly sales trends
- Top 5 product categories by revenue
- Category-wise monthly performance
- Payment method sales distribution
- Loyal customers (more than 10 purchases)
- KPI metrics (Total Sales, Transactions, AOV, Unique Customers)

## Power BI Dashboard
The Power BI dashboard provides:
- KPI cards for high-level performance
- City-wise sales breakdown
- Monthly sales trend analysis
- Top product categories by revenue
- Category-wise sales trend comparison
- Payment method distribution
- Interactive slicers for City, Month, and Payment Method

### Dashboard Name:
**Retail Sales Performance Dashboard**

## Key Insights
- Strong overall revenue with stable monthly performance
- Electronics and Beauty categories drive the majority of sales
- Urban cities contribute the highest revenue
- Digital payment methods dominate transactions
- Loyal customers play a significant role in revenue generation

## Tools & Technologies
- **SQL (MySQL)** – Data analysis and aggregation
- **Power BI** – Data visualization and dashboarding
- **Excel / CSV** – Data source

## Conclusion
This project demonstrates the effective use of SQL and Power BI to transform raw retail transaction data into meaningful business insights. The dashboard is suitable for academic submission, portfolio presentation, and interview discussions.
